# H5 考生端页面截图操作说明

> 文件：`prototype-h5.html`，浏览器打开后按以下步骤逐页操作并截图。
> 推荐使用 Chrome DevTools（F12）的设备模拟模式，或直接对手机模拟框区域截图。

---

## 底部 Tab 结构

| Tab | 页面 | 是否需要登录 |
|-----|------|:----------:|
| 练习题库 | 练习首页 | 否 |
| 技能认证 | 技能选择页 | 是 |
| 我的执照 | 执照申领页 | 是 |
| 个人中心 | 个人中心页 | 是 |

## 截图 1：练习首页

## 快速截图路径汇总

| 序号 | 页面名称 | 操作路径 |
|:----:|---------|---------|
| 1 | 练习首页 | 打开页面（默认） |
| 2 | 题库练习页 | 首页 → 开始做题 |
| 3-6 | 练习答题页 | 题库 → 全部练习 → 作答/答题卡 |
| 7 | 练习结果页 | 完成所有练习题 |
| 8-9 | 模拟考试 | 首页 → 模拟考试 → 作答/交卷 |
| 10-11 | 登录页 | Tab 技能认证（未登录） |
| 12-17 | 信息完善 | 新手机号登录 → 5 步引导 |
| 18 | 技能选择页 | Tab 技能认证（已登录） |
| 19 | 支付页 | 技能选择 → 点击未支付技能 |
| 20 | 科目列表 | 技能选择 → 点击已支付技能 |
| 21-22 | 正式考试 | 科目列表 → 开始考试 |
| 23-26 | 考试结果 | 交卷后（通过/未通过/全科通过） |
| 27-29 | 执照申领 | Tab 我的执照（不同状态） |
| 30 | 考试记录 | 个人中心 → 考试记录 |
| 31-34 | 个人中心 | Tab 个人中心 → 各子页面 |
| 35-36 | 弹窗 | 退出登录 / 支付失败 |
