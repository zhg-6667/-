# 执照认证系统 - 一期 MVP 需求文档

> 基于《执照认证系统一期规划》细化的需求文档，表格驱动风格，涵盖每个页面的字段规格、交互规则、业务规则和异常处理。

---

## 后台模块

### 模块A：题库管理

#### A1. 分类管理页

##### 列表字段

| 字段 | 说明 | 显示规则 |
|------|------|----------|
| 分类名称 | 分类名称 | - |
| 题目数量 | 该分类下生效题目数 | 实时统计 |
| 排序 | 排序权重 | 数字，越小越靠前 |
| 状态 | 启用/停用 | 开关样式 |
| 操作 | 编辑、删除 | - |

##### 新建/编辑弹窗

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 分类名称 | 文本输入 | 是 | 最长50字符，不可与已有分类重名 |
| 排序 | 数字输入 | 是 | 非负整数，默认0 |
| 状态 | 开关 | 是 | 默认启用 |

##### 交互规则
- 列表按排序值升序展示
- 编辑后即时生效

##### 业务规则
1. 删除时校验：该分类下存在题目则不可删除，提示"该分类下存在题目，请先移除或更改分类"
2. 停用分类：已有题目不受影响，但前端练习页不再展示该分类入口
3. 分类名称不可重复

#### A2. 题目列表页

##### 列表字段

| 字段 | 说明 | 显示规则 |
|------|------|----------|
| 题目编号 | 自动生成 | 格式 Q00001 |
| 题目内容 | 题干文本 | 截断前50字 |
| 类型 | 单选/多选/判断 | Tag 样式 |
| 分类 | 所属分类名称 | - |
| 分值 | 题目分值 | - |
| 创建人 | 创建者名称 | - |
| 创建时间 | - | yyyy-MM-dd HH:mm |
| 状态 | 生效/失效 | 开关样式 |
| 操作 | 编辑、删除、启用/禁用 | - |

##### 搜索筛选

| 条件 | 类型 | 说明 |
|------|------|------|
| 关键词 | 文本输入 | 模糊搜索题目内容 |
| 分类 | 下拉选择 | 全部 / 各分类 |
| 状态 | 下拉选择 | 全部 / 生效 / 失效 |
| 时间段 | 日期范围选择器 | 创建时间范围 |

##### 操作按钮
- **编辑**：跳转编辑页，回填所有字段
- **删除**：二次确认弹窗，已被考试引用不可删除
- **启用/禁用**：切换状态，即时生效

##### 业务规则
1. 默认按创建时间倒序，每页20条
2. 删除为逻辑删除（标记失效）
3. 已被考试引用（exam_question 中存在关联）的题目不可删除，提示"该题目已被考试引用，无法删除"

#### A3. 新建/编辑题目页

##### 表单字段

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 题目类型 | 单选按钮组 | 是 | 单选 / 多选 / 判断，选定后不可更改（编辑时禁用） |
| 题目内容 | 富文本编辑器 | 是 | 不可为空 |
| 选项列表 | 动态表单项 | 是 | 单选/多选：2~6个选项；判断：固定"对/错"两项 |
| 选项内容 | 文本输入 | 是 | 每个选项不可为空 |
| 正确答案 | 勾选 | 是 | 单选：勾选1个；多选：勾选≥2个；判断：勾选1个 |
| 分类 | 下拉选择 | 是 | 从分类管理动态获取，仅显示启用状态的分类 |
| 分值 | 数字输入 | 是 | >0，默认1，支持小数（如0.5） |
| 知识点解析 | 富文本编辑器 | 否 | 答错时展示给学员 |
| 题目媒体 | 文件上传 | 否 | 支持图片(jpg/png ≤5MB)、视频(mp4 ≤10MB ≤60秒)、GIF(≤2MB)，最多上传1个 |

##### 交互规则
- 切换题目类型时，选项列表自动重置
- 判断题：选项固定为"对""错"，不可增删
- 单选/多选题：可动态增删选项，点击"添加选项"新增一行
- 编辑模式下，题目编号自动显示且不可编辑
- 媒体上传后显示预览（图片显示缩略图，视频显示播放器，GIF显示动图）
- 答题界面中媒体固定展示在题干文字描述下方

##### 业务规则
1. 保存时校验正确答案：单选必须且仅勾选1个，多选至少勾选2个
2. 保存成功后跳转回列表页，列表自动刷新
3. 编辑已被考试引用的题目时，给出提示"该题目已被考试引用，修改将影响考试内容"

#### A4. 批量导入/导出

##### 导入流程

| 步骤 | 说明 |
|------|------|
| 1. 下载模板 | 点击"下载模板"获取 Excel 模板文件 |
| 2. 填写数据 | 按模板格式填写题目数据 |
| 3. 上传文件 | 点击"批量导入"选择 Excel 文件上传 |
| 4. 校验反馈 | 系统校验后展示导入结果：成功N条，失败M条 |
| 5. 错误下载 | 失败记录可下载错误明细 Excel（含失败原因） |

##### 模板字段

| 列 | 说明 | 校验规则 |
|----|------|----------|
| 题目内容 | 题干文本 | 必填 |
| 题目类型 | 单选/多选/判断 | 必填，必须为指定值 |
| 选项A | 选项A内容 | 必填 |
| 选项B | 选项B内容 | 必填 |
| 选项C | 选项C内容 | 单选/多选必填，判断可空 |
| 选项D | 选项D内容 | 可空 |
| 正确答案 | 如 A 或 A,B,C | 必填，必须在选项范围内 |
| 分类 | 分类名称 | 必填，必须为已存在的分类名 |
| 分值 | 数字 | 必填，>0 |
| 知识点解析 | 文本 | 可空 |

##### 导出
- 按当前筛选条件导出 Excel，字段同模板
- 文件名：题目导出_yyyy-MM-dd.xlsx

### 模块B：考试管理

#### B1. 考试配置（Tab1）

##### 列表字段

| 字段 | 说明 | 显示规则 |
|------|------|----------|
| 考试名称 | 如"科目一" | - |
| 关联课程 | 被哪些课程关联 | 多个课程以逗号分隔 |
| 题目数 | 试卷题目总数 | - |
| 总分 | 满分 | - |
| 及格分 | 及格线 | - |
| 时长 | 考试时长 | 如"30分钟" |
| 重考次数 | 最大重考次数 | - |
| 组卷方式 | 手动选题/随机抽题 | Tag 样式 |
| 切屏容忍次数 | 防作弊切屏阈值 | 数字，如"2次" |
| 状态 | 启用/停用 | 开关样式 |
| 操作 | 编辑、查看详情 | - |

##### 新建/编辑考试

**基础信息：**

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 考试名称 | 文本输入 | 是 | 最长100字符 |
| 考试描述 | 多行文本 | 否 | 最长500字符 |
| 总分 | 数字输入 | 是 | >0 |
| 及格分 | 数字输入 | 是 | >0 且 ≤ 总分 |
| 考试时长 | 数字输入 | 是 | 单位：分钟，>0 |
| 重考次数 | 数字输入 | 是 | ≥0，0表示不允许重考 |
| 组卷方式 | 单选 | 是 | 手动选题 / 随机抽题 |
| 切屏容忍次数 | 数字输入 | 是 | ≥0，默认2，0表示不启用切屏检测 |
| 状态 | 开关 | 是 | 默认启用 |

**手动选题模式：**

| 操作 | 说明 |
|------|------|
| 从题库选题 | 弹窗展示题目列表，支持按分类筛选，勾选后添加 |
| 题目排序 | 拖拽排序，决定试卷中的题目顺序 |
| 移除题目 | 从当前试卷中移除 |
| 自动计算 | 题目数和总分根据选中题目自动累加 |

**随机抽题模式：**

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 分类 | 下拉选择 | 是 | 从分类管理获取 |
| 抽题比例 | 数字输入 | 是 | 百分比值（如40表示40%），各分类比例之和≤100% |

- 可添加多条抽题规则（每条对应一个分类和百分比）
- 各分类比例之和 ≤ 100%，剩余比例的题目从未配置比例的其他分类中随机抽取
- 计算出现小数时向下取整，差额计入剩余随机池
- 后台配置时校验：某分类题目数不足时页面警告提示但不拦截保存（题目后续可能增减）
- 实际抽题时某分类题目不足则有多少抽多少，差额从其他分类补齐
- 每次考生开考时实时抽题

##### 业务规则
1. 已有考试记录的考试不可切换组卷方式，提示"已有考生参加过此考试，无法切换组卷方式"
2. 随机抽题模式下，系统根据配置的分类比例和总题目数计算各分类抽题数量
3. 停用考试后，前端不再展示该科目，已报考考生不受影响

### 模块D：认证审核

#### D1. 审核列表页

##### 筛选标签
全部 / 待审核 / 已通过 / 不通过（Tab 切换，显示各状态数量角标）

##### 列表字段

| 字段 | 说明 | 显示规则 |
|------|------|----------|
| 考生姓名 | - | - |
| 手机号 | - | - |
| 材料类型 | 身份证正面/反面/个人照片 | Tag 样式 |
| 提交时间 | - | yyyy-MM-dd HH:mm |
| 状态 | 待审核/已通过/不通过 | 彩色 Tag |
| 材料预览 | 缩略图 | 点击可放大查看 |
| 操作 | 通过、不通过 | 待审核状态显示 |

##### 审核操作

**通过：**
- 点击"通过"，二次确认后状态变更为已通过
- 该考生的 user_material.audit_status 更新为1
- 如该考生所有材料（身份证正反面 + 个人照片）均审核通过，自动将 enrollment.audit_status 更新为已通过

**不通过：**
- 点击"不通过"，弹窗填写不通过原因（必填，最长500字符）
- 考生端显示不通过原因，引导重新上传

##### 业务规则
1. 只有待审核状态的材料才显示操作按钮
2. 已通过/不通过的记录可查看但不可再操作
3. 代报名直接标记通过的材料，审核列表中状态显示为"已通过"，审核备注显示"代报名免审"

## 移动端模块

### 模块1：用户登录/注册

#### 1.1 登录页

##### 页面字段

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 手机号 | 数字输入 | 是 | 11位手机号格式校验 |
| 图形验证码 | 文本输入 | 是 | 4位字母/数字 |
| 短信验证码 | 数字输入 | 是 | 6位数字，5分钟内有效 |

##### 交互规则
- 图形验证码：右侧显示验证码图片，点击可刷新
- "获取验证码"按钮：图形验证码验证通过后可点击，点击后60秒倒计时，倒计时内不可重复点击
- 手机号格式不正确时，输入框下方提示"请输入正确的手机号"
- 新手机号自动注册，无需单独注册流程
- 登录成功后：
  - 新用户（is_info_completed=0）→ 跳转信息完善页
  - 老用户 → 返回触发登录的页面

##### 异常处理
- 验证码错误 → 提示"验证码错误，请重新输入"，清空验证码输入框
- 验证码过期 → 提示"验证码已过期，请重新获取"

##### 业务规则
1. 每次切题时实时保存答案到服务端（调用 /api/exam/answer）
2. 开始考试后在 exam_record 创建记录，记录 started_at
3. 交卷时计算得分、正确数、错误数，更新 exam_record
4. 交卷后检查该课程下所有科目通过状态，如全部通过则更新 enrollment.exam_status = 2

| 元素 | 说明 |
|------|------|
| 头部信息卡 | 头像（圆形）、姓名、手机号（中间4位隐藏：138****1234） |
| 功能列表 | 账号信息、认证材料、考试记录、退出登录 |

##### 功能列表

| 入口 | 跳转 | 说明 |
|------|------|------|
| 账号信息 | 账号信息编辑页 | 右侧箭头 |
| 认证材料 | 材料状态页 | 右侧显示审核状态摘要 |
| 考试记录 | 模块5成绩列表页 | 右侧箭头 |
| 退出登录 | 清除登录状态，跳转登录页 | 红色文字，二次确认 |

---

#### 6.2 账号信息编辑页

##### 表单字段

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 头像 | 图片上传 | 否 | 支持 jpg/png，≤5MB，圆形裁剪 |
| 昵称 | 文本输入 | 是 | 最长50字符 |
| 姓名 | 文本输入 | 是 | 最长50字符 |
| 手机号 | 文本（只读） | - | 不可修改 |

##### 交互规则
- 修改后点击"保存"按钮提交
- 保存成功提示"保存成功"，返回个人中心

---

#### 6.3 认证材料状态页

##### 页面元素

| 材料 | 显示内容 |
|------|----------|
| 身份证正面 | 缩略图 + 审核状态（未上传/待审核/已通过/不通过） |
| 身份证反面 | 缩略图 + 审核状态 |
| 个人照片 | 缩略图 + 审核状态 |

##### 交互规则
- 未上传 → 显示上传入口
- 待审核 → 显示缩略图 + "审核中"标签，不可操作
- 已通过 → 显示缩略图 + 绿色"已通过"标签
- 不通过 → 显示缩略图 + 红色"不通过"标签 + 不通过原因 + "重新上传"按钮
- 重新上传后状态重置为待审核
