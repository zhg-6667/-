# 26年3月 - 执照认证系统
执照认证系统账号信息、考试模块的设计

## 执照认证系统规划

### 对象

- 学员
- 培训师

---
## 主流程
打开App，默认进入题库
├── 练习题库（免登录）
│   ├── 练习首页
│   │   ├── 顶部进度卡片：做题进度环形图、做题数、累计刷题、正确率
│   │   ├── "开始做题"按钮
│   │   ├── 快捷入口：随机练习 | 错题回顾 | 模拟考试
│   │   ├── 科目考试信息卡片：题目总数、模拟考试及格线、正式考试及格线及次数限制
│   │   │   └── "开始模拟考试"按钮
│   │   └── 知识点·场景学习：按场景分类的学习卡片（如厨房清洁、宠物毛发处理等）
│   ├── 题库练习页
│   │   ├── 统计概览：总题数 | 已答题 | 未做题
│   │   ├── "全部练习"按钮
│   │   └── 按知识点练习：职业认知与规范 | 文化清洁 | 业务场景 等分类
│   ├── 答题页
│   │   ├── 题目类型：单选（单选按钮）| 多选（勾选框）| 图片题（图片+选项）| 判断题
│   │   ├── 答题后即时反馈：
│   │   │   ├── 回答正确 → 绿色提示，自动进入下一题
│   │   │   └── 回答错误 → 红色提示 ✗ + 正确答案 + 你的答案 + 知识点解析
│   │   ├── 底部状态栏：正确数 · 错误数 · 当前题号/总题数
│   │   └── 答题卡面板：题号网格，已答/未答/正确/错误用不同颜色区分
│   └── 练习结果页
│       ├── "本次练习完成" + 图标
│       ├── 统计：总答题数 | 错误数 | 正确率
│       └── 操作按钮：回顾错题 | 继续答题
├── 技能报考认证
│   ├── 未登录 → 登录页
│   │   ├── 有账号 → 手机验证码登录 → 登录成功 → 返回
│   │   └── 无账号 → 手机验证码注册 → 注册成功 → 引导信息完善流程
│   │       └── 账号信息录入（头像/昵称/个人照片）
│   │         → 身份证上传（正反面，OCR识别）
│   │         → 健康证上传（OCR识别有效期）
│   │         → 体检报告上传
│   │         → 背调报告上传
│   │         → 完善完成 → 返回原页面
│   ├── 选择技能 → 未支付 → 支付认证费
│   │               ├── 支付成功 → 进入科目列表
│   │               └── 支付失败 → 重试/返回
│   └── 已支付 → 科目列表（科目一~四）
│       └── 开始考试 → 答题页
│           ├── 题目类型：单选 | 多选 | 图片题 | 判断题（同练习题库）
│           ├── 顶部倒计时（30min），超时自动交卷
│           ├── 答题过程中不显示对错，仅记录答案
│           ├── 答题卡面板：题号网格，已答/未答颜色区分
│           └── 交卷 → 考试结果页
│               ├── 通过 → 考试名称 + 大号绿色分数 + 总题数 + 错误数
│               │       + "恭喜你完成了本次考试！" + "查看证书"按钮
│               ├── 未通过 → 考试名称 + 大号橙色分数 + 总题数 + 错误数
│               │         + "考试不通过，建议回顾知识后再来考试"
│               │   ├── 剩余重考次数 > 0 → "重新考试"按钮
│               │   └── 剩余重考次数 = 0 → 支付重考费 → 支付成功 → 重置重考次数 → 重考
│               │                         └── 支付失败 → 重试/返回
│               └── 科目一~四全部通过 → 提示进入执照申领流程
├── 执照申领（需登录）
│   ├── 前置条件：科目一~四全部通过 + 认证材料审核通过 + 头像已上传
│   ├── 条件未满足 → 展示缺失项，引导补全
│   └── 条件满足
│       ├── 未开通过电子执照 → 支付工本费
│       │   ├── 支付成功 → 开通电子执照 + 显示"实体执照制作中"
│       │   └── 支付失败 → 重试/返回
│       └── 已开通电子执照 → 查看执照详情
│           ├── 执照详情：执照编号、姓名、技能名称、发证日期、有效期、电子执照卡片
│           └── 申请重做（实体执照）→ 支付工本费 → 支付成功 → 进入实体执照制作流程
│                                                   └── 支付失败 → 重试/返回
├── 考试记录 → 历史成绩列表 → 答题详情
└── 我的 → 账号信息/证件信息/认证材料

## 信息模块

### 账号注册 & 登录

1. **登录验证**
   - 访问内部页面的时候要先判断是否登录，未登录的话，要跳转到登录页面

2. **协议**
   - 点击后展示协议信息

3. **微信授权登录**
   - 接收微信授权
   - 记录微信授权
   - 创建账号
     - 以手机号码为唯一判断，如手机未注册过账号，则自动注册
     - 提前设计账号中心，可以根据不同平台创建账号，不同平台如：微信、好慷在家
   - 页面元素调整
     - 图标
     - 文案显示：执照认证平台

4. **手机验证码登录/注册**
   - 图形验证码
     - 点击后更换图片数字
     - 验证通过后获取验证码按钮可以点击
   - 验证码
     - 输入验证码后可以点击提交
     - 点击提交
       - 如果协议没有勾选，弹窗提示确认勾选，点击确认后，默认勾选
       - 验证通过：进行登录，并缓存登录状态
       - 验证失败：验证异常，请重试

5. 登录成功后跳转到触发登录的页面

### 支付中心

> 对接微信支付

- 接入微信支付
- 订单提交页
- 支付成功/失败页面
- 申请退款

### 个人中心

- **账号信息**
  - 头像、 昵称、手机号
  - 个人照片（后续用于监考场景人脸识别，先不拓展）：考生自己上传头像，点击确认上传后，使用AI接口进行人像验证，验证失败提示具体原因（如"未检测到人脸"、"检测到多人"、"照片模糊"），引导重新上传
- **证件信息**
  - 健康证、身份证，上传图片后OCR自动识别有效期及姓名、证件号，与账号信息交叉校验（防止上传他人证件）。如果有效期到了，提示过期。任何时候都可以重新上传证件信息
- **认证**
  - 体检报告：用户上传，管理员审核，显示体检结果：待审核/通过/不通过
  - 背调：用户上传，管理员审核，背调结果：待审核/通过/不通过，不通过的话，显示原因

## 用户管理中心

### 管理员账号中心

- 顶部显示当前登录人员的：姓名、id

### 考生列表

- **基本信息：**
  - 账号id、头像、昵称、姓名、执照编号(省份编码+城市编码+执照编码)、状态（有效、失效）、微信、个人照片、手机、城市
- **操作：**
  - 添加考生
  - 编辑
  - 考试管理
    - 点击后跳转到该考生的考试管理页面

### 认证审核

- 待审核列表：展示考生提交的体检报告、背调材料，状态为"待审核"
- 操作：通过 / 不通过（不通过需填写原因）
- 审核完成后，考生端认证状态同步更新

### 添加考生

- 弹窗编辑，字段参照基本信息

### 编辑

- 弹窗编辑，字段参照基本信息

### 技能配置（TODO：后续补充）

- 技能的创建、编辑、删除
- 技能关联的考试配置

### 题库管理（TODO：后续补充）

- 考试题目的录入、编辑
- 组卷规则

### 考试管理
> 认证什么技能、有哪些考试、处于什么状态、考试的内容（选择）
- 技能列表
  - 分页显示所有认证技能
  - 每个技能包含多个考试，考试状态：待考试、考试中、考试通过、考试不通过
  - 待考试状态下，考生自行点击"开始考试"进入答题，有时长限制（通过配置设置），超时自动提交
  - 点击后弹窗显示考试内容
  - 考试不通过可重考，重考次数通过配置控制
  - 技能下科目一与科目四考试通过后获得执照（执照发放逻辑在其他模块设计）
  - 考试内容：展示每项考试的具体考试题目，选择的答案，是否正确，最终得分

### 考生端（学员视角）

- **技能认证进度**：展示考生所有技能的认证状态及各科目完成情况
- **进入考试**：在待考试的科目中点击"开始考试"进入答题页面
- **答题页面**：显示题目、倒计时，支持手动提交和超时自动提交
- **历史成绩**：查看每次考试的成绩记录、答题详情
