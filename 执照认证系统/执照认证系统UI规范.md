## 4. 组件规范

### 4.1 按钮 Button

#### 后台按钮

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主按钮 `.btn-primary` | 蓝底白字，hover 变亮 `#4096FF` | 新建、确认、搜索 |
| 成功按钮 `.btn-success` | 绿底白字 | 通过、激活 |
| 危险按钮 `.btn-danger` | 红底白字 | 删除、吊销 |
| 默认按钮 `.btn-default` | 白底灰边，hover 变蓝边蓝字 | 取消、次要操作 |
| 链接按钮 `.btn-link` | 无背景蓝字 | 表格行内操作（编辑） |
| 链接按钮-橙 `.btn-link-orange` | 无背景橙字 | 查看详情 |
| 链接按钮-绿 `.btn-link-green` | 无背景绿字 | 审核通过 |
| 链接按钮-红 `.btn-link-red` | 无背景红字 | 删除、拒绝 |
| 小按钮 `.btn-sm` | 高度 28px，字号 12px | 紧凑场景 |

**通用样式：** 高度 36px，圆角 6px，内边距 0 16px，点击时 `scale(0.97)`

#### H5 按钮

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主按钮 `.btn-primary` | 渐变蓝底白字，带阴影 | 开始做题、提交、交卷 |
| 描边按钮 `.btn-outline` | 白底蓝边蓝字 | 继续答题、次要操作 |

**通用样式：** 高度 46px，圆角 23px（全圆角胶囊），宽度 100%，点击时 `scale(0.97) + opacity(0.9)`

## 5. 页面动画

| 场景 | 动画 |
|------|------|
| 页面切换 | `fadeIn` — 从下方 4~6px 位移 + 透明度渐入，0.25s |
| 弹窗打开 | 缩放 `scale(0.85/0.92) → 1` + 透明度，0.2~0.25s |
| 底部面板 | `translateY(100%) → 0`，0.3s |
| 按钮点击 | `scale(0.97)`，瞬时 |
